# alx_backend_capstone
 ALX CAPSTONE PROJECT
MySika Budget is a Django-powered personal finance management and investment tracking application designed to help users take full control of their money. The app allows individuals to track daily expenses, incomes, budgets, savings goals, and monitor both Ghanaian (GSE) and US stock markets in one unified platform.

Built as an ALX Backend Final Capstone Project, MySika Budget demonstrates modern backend engineering using:
Python & Django
Django REST Framework (DRF)
JWT/Token Authentication
External financial APIs for stock/market data

 Core Features
1. Expense & Income Tracking
Record transactions, categorize spending, and visualize financial habits.

2. Budget Management
Set monthly/weekly budgets and track progress in real time.

3. Savings Goals
Create and monitor savings projects with milestone tracking.

4. Ghanaian & US Stock Tracking
Fetch stock prices from GSE and US markets using external APIs.

5. User Accounts & Authentication
Secure login, registration, and token-based authentication.

Reports & Analytics
Summaries, breakdowns, and insights to guide better financial decisions.
